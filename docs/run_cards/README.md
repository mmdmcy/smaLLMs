# Run Cards

This directory is reserved for curated, human-readable run cards from real
smaLLMs benchmark runs.

Only commit run cards that meet these conditions:

- generated by the maintained CLI-first runner
- produced from a clean git tree
- based on real local model execution
- clear about benchmark selection and sample count
- small enough to review in a pull request

Generated machine artifacts should remain under `artifacts/`, which is ignored by
git by default. For portfolio use, commit a small number of representative
`RUN_CARD.md` files here after renaming them to include the model and suite, for
example `qwen3_5_piqa_25_samples.md`.
